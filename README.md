MdConnector
===========

Markdown Connector

## 소개

마크다운을 위한 탐색기용 연결프로그램. 웹페이지와 연동된 Markdown 파일로 바로 이동하기 위한 연결프로그램 

## 사용법

1. 마크다운 파일이 존재하는 Dropbox 폴더명을 연결할 웹페이지로 변경 : wiki.dataknow.net
2. 탐색기 => 연결프로그램 => 선택 => MdConnector.exe 선택
